ClientInfo
	comprises
		- Application
		- ApplicationVersion
		- ApplicationSource
		- ApplicationUser
		- Driver
		- DriverInfo
		- DriverVersion

	of these, hdbconnect sets initially
		- Application
		- Driver
		- DriverInfo
		- DriverVersion

	is sent to server only after authentication (and only if it was changed)	

	values appear in M_SESSION_CONTEXT
